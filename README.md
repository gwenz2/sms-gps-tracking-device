# SIM800TEST

A simple GPS tracker project using ESP8266, SIM800L, and NEO-6M.

![Wiring Screenshot](screenshot.png)

## Features
- Receives SMS with the command `CHECK` and replies with GPS location.
- Uses SIM800L for GSM communication and NEO-6M for GPS parsing.
